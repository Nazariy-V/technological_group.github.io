Brief description of project one.

Starbucks

Цей проект представляє повністю розроблене меню гарячих напоїв, що поєднує HTML-структуру та CSS-стилістику для створення цікавого та інтерактивного користувацького досвіду. Він демонструє основні методи веб-розробки, роблячи навігацію по меню плавною і візуально привабливою.

Sweety Shop

Цей проект - повністю розроблений веб-сайт електронної комерції для Sweety Shop, створений за допомогою HTML і CSS для створення інтерактивної та візуально привабливої вітрини інтернет-магазину.Він включає в себе структуровану навігацію, меню продуктів, систему відгуків та інтегровані посилання на соціальні мережі.
